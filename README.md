# Dashboard-Northwind

Análisis de ventas de la empresa Northwind desarrollado en Power BI.

## ¿De qué trata?

Dashboard interactivo que muestra el desempeño comercial de Northwind,
con filtros por país y año para explorar la información.

## ¿Qué incluye?

- Resumen general de ventas, pedidos, unidades y países
- Ventas por mes
- Top 5 productos más vendidos
- Mapa de países clientes
- Ventas por categoría de producto
- Detalle de pedidos por cliente

## Herramienta

Power BI Desktop
